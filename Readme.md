# 🚀 AI-Powered Interview Preparation Platform

An intelligent full-stack web application that helps candidates prepare for interviews using AI-driven insights, personalized roadmaps, resume analysis, and mock interview evaluations.

---

## 📌 Overview

Preparing for interviews often requires identifying skill gaps, practicing relevant questions, and tailoring resumes for specific roles. This platform automates that process by analyzing a candidate's **resume** and **job description** to generate a personalized interview preparation experience.

The system helps candidates understand their readiness level, improve weak areas, and simulate real interview scenarios with AI-generated feedback.

## 🏗️ System Workflow

1. User creates an account and logs in securely.
2. Candidate uploads their resume.
3. Candidate provides the target job description.
4. AI analyzes both inputs.
5. System generates:

   * Interview Questions
   * Skill Gap Analysis
   * Overall Readiness Score
   * One Week Preparation Roadmap
6. Candidate starts an AI Mock Interview.
7. AI evaluates responses and generates a detailed report.
